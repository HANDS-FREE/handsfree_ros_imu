# handsfree_ros_imu 功能包
![](./tutorials/img/ppt/1.jpg)    

## 介绍

handsfree_ros_imu 是用于学习使用 IMU 的 ROS 功能包，适用于 handsfree 推出的 3 款 ROS IMU。同时也含有纯 python 编写的输出数据显示, 支持 Linux、Windows 相关的脚本文件。
